Our Mission

TruthLens aims to reduce misinformation and provide accessible, AI-powered tools to verify media and claims. We combine AI analysis (Google Gemini), reverse-image checks, and human-friendly explanations to help people make informed decisions.

How It Works

  1. User submits text, link, image, or video.
  2. Server composes a structured prompt for Gemini and additional services (reverse image search, transcript extraction).
  3. Gemini returns an analysis which we parse into a credibility score, summary, and sources.
  4. Results are saved to the user's history and can be exported or shared.

Privacy

We store only data required for user features and user history. Uploaded media is stored temporarily and can be deleted by the user. DO NOT upload sensitive personal data.
